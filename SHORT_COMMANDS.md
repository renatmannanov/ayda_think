# Короткие команды

## 1. Отправить изменения в Git (Deploy на Railway)
Выполните эти команды по очереди в терминале:
```bash
git add .
git commit -m "Ваше сообщение"
git push
```
*После `git push` Railway автоматически начнет сборку и деплой.*

## 2. Запустить сервер (Локально)
**Вариант А (если есть Git Bash):**
```bash
bash start.sh
```

**Вариант Б (Windows PowerShell / CMD):**
Нужно запустить два процесса. Откройте два терминала.

Терминал 1 (API Сервер):
```powershell
python api_server.py
```

Терминал 2 (Бот):
```powershell
python main.py
```

## 3. Остановить сервер
Нажмите `Ctrl+C` в терминале, где запущен процесс.
Если процессов два — нажмите `Ctrl+C` в каждом из них.
