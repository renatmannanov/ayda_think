# Пет-проект: ayda_think — AI для поиска смыслов и закономерностей

## Суть идеи

Мысли как **кристаллы** — собираются в моменте, разрозненно, без контекста.
Цепочки мыслей как **артефакты** — паттерны которые человек сам может не увидеть.

AI не создаёт контент. AI **находит связи** между мыслями которые ты уже записал — через время, через теги, через смысл — и показывает картину которую ты не мог увидеть сам, потому что смотрел на пазлы по одному.

```
Мысль в 2:00 ночи           →  кристалл
Заметка из saved TG         →  кристалл  
Обрывок из приватного канала →  кристалл
                                    ↓
                          AI находит паттерн
                                    ↓
                    "Вот цепочка из 7 мыслей за 3 месяца.
                     Ты думаешь об одном и том же разными словами.
                     Вот что из этого может вырасти."
```

---

## Контекст и мотивация

Из разговора с Айгерим:
> "Мне нужен большой поток информации и мыслей, чтоб сгенерировать основу, истину и осознать её"
> "Пока в потоке мыслей — ощущаю тревогу от того, что истина где-то рядом"

Это не проблема записи. Это проблема **синтеза**. Мыслей много, они записываются — но связь между ними теряется потому что мозг не держит всё одновременно.

AI может держать всё одновременно и находить то что мозг пропускает.

---

## Три уровня задачи

**Уровень 1 — Не потерять кристалл**
Мысль появилась → быстро записал в Telegram → она проиндексирована и никуда не денется. Даже если забудешь.

**Уровень 2 — Найти цепочку**
AI периодически проходит по всей базе и находит кластеры связанных мыслей. Не по тегам — по смыслу. Мысль из января и мысль из октября могут быть об одном.

**Уровень 3 — Родить артефакт**
Когда цепочка найдена — AI предлагает синтез. Не пост. Не резюме. А **следующую мысль** которую ты сам ещё не додумал. Достроенный образ.

## Архитектура

```
Источники (приватный TG + saved messages + Айгерим если захочет)
              ↓
    Telegram API (Telethon) — читает новые сообщения раз в день
              ↓
    Чанкинг — каждое сообщение отдельный кристалл
    + метаданные: дата, теги (#ayda_think, #idea и т.д.), автор
              ↓
    Эмбеддинги → Chroma (локально на ноутбуке)
              ↓
    ┌─────────────────────────────────────┐
    │  Два режима работы:                 │
    │                                     │
    │  ПАССИВНЫЙ (фоновый)                │
    │  Раз в неделю — кластеризация       │
    │  Находит новые цепочки              │
    │  Присылает алерт в TG               │
    │                                     │
    │  АКТИВНЫЙ (по запросу)              │
    │  "Что я думал про X?"               │
    │  "Найди связи между этим и этим"    │
    │  "Что я думал об этом 6 месяцев"    │
    └─────────────────────────────────────┘
              ↓
    LLM (DeepSeek 14B, локально)
    Синтезирует цепочку → предлагает артефакт
```

## Что AI делает конкретно

### Поиск паттернов во времени
```
Запрос: "покажи как менялась моя мысль про IAM и AI"

Ответ:
→ Март 2024: "интересно было бы автоматизировать аудит прав"
→ Июль 2024: "если модель видит всю матрицу — она найдёт то что я пропускаю"  
→ Январь 2025: "SoD конфликты можно детектить через граф"
→ Февраль 2025: "AI ассистент для IAM — реальный продукт"

Паттерн: мысль созревала 11 месяцев. Каждый раз новый угол.
Следующий шаг который ты ещё не записал: [...]
```

### Поиск неочевидных связей
```
Запрос: "что общего между мыслями про бег и мыслями про бизнес?"

Ответ: в обоих случаях ты возвращаешься к теме 
"системность vs интуиция". В беге — план тренировок vs 
слушать тело. В бизнесе — процессы vs живой контакт.
Это одна мысль в двух контекстах.
```

### Детектирование "горячих тем"
```
Пассивный алерт:
"За последние 2 недели ты записал 8 мыслей так или иначе 
связанных с темой [локальный AI + конфиденциальность]. 
Это больше чем обычно. Хочешь посмотреть цепочку?"
```

### Достройка артефакта
```
Запрос: "вот эти 5 мыслей — что из них может вырасти?"

Ответ: не резюме, а следующий шаг который логично следует 
из цепочки но который ты ещё не сформулировал.
```

## Совместный режим (с Айгерим)

Это отдельная и интересная возможность. Два человека думают об одном — но видят разное.

Вариант реализации:
- У каждого своя база кристаллов (приватная)
- Можно добровольно "шерить" конкретный кластер
- AI находит пересечения: "вы думаете об одном разными словами"
- Или наоборот: "вы думаете по-разному об одном — вот в чём расхождение"

Это не про создание совместного контента. Это про **понимание как два человека думают** — где синхронизируются, где расходятся, где один дополняет другого.

Технически просто: две отдельные Chroma коллекции + запрос к обеим при поиске пересечений.

### Кластеризация (еженедельная)
```python
from sklearn.cluster import DBSCAN
import numpy as np

def find_chains():
    # Получаем все эмбеддинги
    data = collection.get(include=['embeddings', 'documents', 'metadatas'])
    embeddings = np.array(data['embeddings'])
    
    # DBSCAN — не требует задавать число кластеров заранее
    clustering = DBSCAN(eps=0.25, min_samples=2, metric='cosine')
    labels = clustering.fit_predict(embeddings)
    
    chains = {}
    for idx, label in enumerate(labels):
        if label != -1:
            chains.setdefault(label, []).append({
                'text': data['documents'][idx],
                'date': data['metadatas'][idx]['date'],
                'tags': data['metadatas'][idx]['tags']
            })
    
    # Сортируем по размеру — большие цепочки интереснее
    return sorted(chains.values(), key=len, reverse=True)
```

### Синтез артефакта через LLM
```python
def synthesize_artifact(chain):
    thoughts = '\n'.join([
        f"[{c['date'][:10]}] {c['text']}" 
        for c in sorted(chain, key=lambda x: x['date'])
    ])
    
    prompt = f"""Вот цепочка мыслей одного человека, записанных в разное время:

{thoughts}

Задачи:
1. Найди что общего — какая одна идея пульсирует через все эти записи?
2. Покажи как мысль менялась или углублялась со временем
3. Предложи следующую мысль которую человек ещё не сформулировал — 
   логичное продолжение цепочки

Не резюмируй. Достраивай."""

    response = ollama.chat(
        model='deepseek-r1:14b',
        messages=[{'role': 'user', 'content': prompt}]
    )
    return response['message']['content']
```


## Почему это правильный фокус

Создание постов — это про выход. Поиск смыслов — это про **понимание себя**.

Разница принципиальная:
- Посты мотивируют только если хочешь публиковать
- Поиск паттернов мотивирует сам по себе — это интересно

И это честнее как инструмент. Не "помоги мне написать", а "помоги мне увидеть что я уже думаю".
