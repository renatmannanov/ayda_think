# Step 5 — Тест на реальных данных + итерация

## Что делаем
Тестируем `/artifact` на реальных кластерах, итерируем промпты.

## План тестирования

### 1. Копируем продовую БД локально
```bash
python scripts/copy_prod_to_local.py
```

### 2. Тестируем /artifact на разных темах
- Тема с большим кластером (15+ фр.): `/artifact чайный бизнес`
- Тема с маленьким кластером (5-7 фр.)
- Тема в шуме (нет кластера)
- Тема на английском
- Абстрактная тема: `/artifact смысл жизни`
- Тема с противоречиями в заметках

### 3. Оценка качества
- [ ] GPT не галлюцинирует (все факты из заметок)
- [ ] Ссылки [#N] корректные
- [ ] Анализ эволюции, а не пересказ
- [ ] "Куда может двигаться мысль" — осмысленно, не generic
- [ ] Работает при 3-5 фрагментах (graceful degradation)
- [ ] Двухпроходная выборка адекватная (не пропускает важное)

### 4. Итерация промптов
- Если пересказывает → усилить "НЕ пересказывай"
- Если галлюцинирует → усилить "ТОЛЬКО из заметок"
- Если слишком длинно → добавить "будь лаконичен"
- Если не цитирует → усилить "каждый вывод = ссылка"

### 5. Тест пассивных артефактов
- Запустить /cluster на локальной БД
- Проверить что артефакты сгенерировались для крупных кластеров
- Оценить качество пассивных vs активных

## Скрипт для локального тестирования
Создать `scripts/test_synthesis.py` — вызывает synthesize() напрямую без бота.
