# Step 3 — /artifact команда

## Что делаем
Команда `/artifact [тема]` в brain_handler.py.

## Логика

1. Пользователь: `/artifact чайный бизнес`
2. Семантический поиск по теме (как /search, но больше результатов — limit=30)
3. Если результат в кластере — подтягиваем все фрагменты кластера (для полноты)
4. Объединяем, дедуплицируем по id, сортируем по дате
5. Вызываем `synthesize(topic, fragments)`
6. Сохраняем артефакт через `save_artifact()`
7. Отправляем пользователю GPT-ответ

## Пайплайн подробнее

```
/artifact чайный бизнес
    │
    ├── 1. Embedding запроса (OpenAI)
    ├── 2. search_hybrid("чайный бизнес", limit=30)
    │       → [frag_1, frag_5, frag_12, ...]
    │
    ├── 3. Кластерный контекст:
    │       frag_1 → cluster_7
    │       frag_5 → cluster_7
    │       frag_12 → noise (нет кластера)
    │       → Кластер 7 имеет 15 фрагментов
    │       → Подтягиваем ВСЕ фрагменты кластера 7
    │
    ├── 4. Объединяем: search results ∪ cluster fragments
    │       → ~35 уникальных фрагментов, sorted by date
    │
    ├── 5. synthesize("чайный бизнес", fragments)
    │       → Проход 1 (если > 30): GPT выбирает 20 ключевых
    │       → Проход 2: GPT анализирует эволюцию
    │
    ├── 6. save_artifact(topic, content, fragment_ids, cluster_id=7)
    │
    └── 7. Отправить пользователю:
            🧬 Артефакт: "чайный бизнес"
            📊 На основе 20 заметок (кластер "Чайный бизнес", 15 фр.)

            [GPT-ответ с ссылками на заметки]
```

## Формат ответа в Telegram

```
🧬 Артефакт: "чайный бизнес"
📊 20 заметок, янв-мар 2026 | 📦 Чайный бизнес (15 фр.)

[GPT-анализ эволюции мысли]

📌 Ключевые точки:
• [#3] 2026-01-15: "Хочу попробовать чай..."
• [#8] 2026-02-03: "Встреча с поставщиком..."
• [#14] 2026-03-01: "Может подарочные наборы?"
```

## Файлы для изменения
- `bot/brain_handler.py` — добавить artifact_command
- `main.py` — зарегистрировать handler

## Edge cases
- Тема не найдена (0 результатов) → "По теме X ничего не найдено"
- Мало фрагментов (< 3) → показать просто список, без синтеза
- GPT-ошибка → показать ошибку пользователю, не молчать
- Длинный ответ GPT (> 4096 символов Telegram) → разбить на части
