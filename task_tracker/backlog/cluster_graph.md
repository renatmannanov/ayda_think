# Граф связей между кластерами

## Идея
Построить граф, где узлы — кластеры, рёбра — связи между ними.

## Типы связей (рёбра)

1. **По тегам**: общие теги между кластерами (напр. #idea + #200k → idea↔200k)
2. **По эмбеддингам**: косинусная близость центроидов кластеров
3. **По "мостовым" фрагментам**: фрагменты с тегами из 2+ кластеров

## Визуализация
- D3.js force-directed graph в HTML
- Или networkx → экспорт в HTML

## Зависимости
- Кластеризация (Этап 4) — готово
- Центроиды кластеров — уже считаются в export_clusters_html.py

## Приоритет
Низкий — полезно для понимания структуры, но не критично.
